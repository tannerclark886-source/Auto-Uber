# Privacy Policy

This Privacy Policy describes how the Auto Uber API project collects and uses information when you use the software.

Effective date: 2025-11-18

## Data Collected
- Location data (pickup and dropoff coordinates) provided by the user or device.
- Minimal device/connection information for debugging (user agent, timestamps) when logging errors.

## Purpose
- To submit ride requests to the Uber Sandbox API on the user’s behalf.
- To log errors and assist with debugging integrations.

## Data Sharing
- We do NOT sell or share personal data with third parties except to the extent required to interact with the Uber API (i.e., Uber receives ride request data you submit).

## Retention
- Data is retained only as long as necessary for debugging and testing. If you request deletion, we will remove your data where practicable.
